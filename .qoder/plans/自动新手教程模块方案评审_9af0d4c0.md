## 四、潜在问题与解决方案

### 4.1 状态转换问题

**问题1: 状态超时处理不够详细**

方案中提到"30秒内未检测到自身角色，判定任务失败"，但未说明:
- 如何处理超时后的重试？
- 重试几次后彻底失败？
- 失败后如何通知用户？

**解决方案**:
```python
class TutorialStateMachine:
    def __init__(self):
        self.max_retries = 3
        self.retry_count = 0
        self.timeout = 30
    
    def handle_timeout(self, state):
        """处理状态超时"""
        self.retry_count += 1
        if self.retry_count >= self.max_retries:
            # 保存截图并通知用户
            self.save_error_screenshot(f"{state}_timeout")
            self.notify_user(f"新手教程失败: {state}超时")
            self.transition_to(States.FAILED)
        else:
            # 重试当前状态
            self.logger.warning(f"{state}超时，重试 {self.retry_count}/{self.max_retries}")
            self.retry_current_state()
```

**问题2: 状态跳转失败恢复机制不明确**

**解决方案**:
```python
# 实现状态回退机制
def transition_to(self, new_state):
    """状态转换（带回退支持）"""
    if not self.can_transition_to(new_state):
        # 尝试回退到前一个稳定状态
        self.rollback_to_stable_state()
        return False
    
    self.previous_state = self.current_state
    self.current_state = new_state
    self.retry_count = 0  # 重置重试计数
    return True
```

### 4.2 图像识别问题

**问题: 图像匹配的鲁棒性**

方案中依赖多个图像匹配（confirm.png、end01.png等），但未考虑:
- 不同分辨率下的适配问题
- 图像匹配失败的备选方案

**解决方案**:
```python
# 多级匹配策略
def find_button(self, button_name, threshold=0.7):
    """查找按钮（多级匹配）"""
    # 1. 尝试模板匹配
    try:
        button = self.find_one(button_name, threshold=threshold)
        if button:
            return button
    except ValueError:
        pass
    
    # 2. 尝试OCR识别
    texts = self.ocr()
    button_text = self.find_boxes(texts, match=re.compile(button_name))
    if button_text:
        return button_text[0]
    
    # 3. 返回None表示未找到
    return None
```

### 4.3 性能问题

**问题: YOLO检测频率过高可能影响性能**

**解决方案**:
```python
class TutorialDetector:
    def __init__(self):
        self.detection_interval = 0.1  # 100ms检测一次
        self.last_detection_time = 0
    
    def should_detect(self):
        """判断是否应该执行检测"""
        current_time = time.time()
        if current_time - self.last_detection_time >= self.detection_interval:
            self.last_detection_time = current_time
            return True
        return False
```
