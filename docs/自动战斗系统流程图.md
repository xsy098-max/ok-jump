# 自动战斗系统流程图

本文档详细描述自动战斗系统的执行逻辑、状态转换和各模块间的交互关系。

## 一、系统架构概览

```mermaid
graph TB
    subgraph 配置层
        A1[AutoCombatTask.json<br/>技能开关/间隔]
        A2[游戏热键配置.json<br/>按键映射]
        A3[基础选项.json<br/>后台模式设置]
    end
    
    subgraph 控制器层
        B1[StateDetector<br/>状态检测器]
        B2[MovementController<br/>移动控制器]
        B3[SkillController<br/>技能控制器]
        B4[DistanceCalculator<br/>距离计算器]
    end
    
    subgraph 工具层
        C1[BackgroundManager<br/>后台管理器]
        C2[PseudoMinimizeHelper<br/>伪最小化助手]
    end
    
    subgraph 检测层
        D1[YOLO模型<br/>fight.onnx]
    end
    
    A1 --> B3
    A2 --> B3
    A3 --> C1
    C1 --> C2
    B1 --> D1
    B1 --> B4
    B2 --> B4
    B3 --> B4
```

## 二、初始化阶段流程

```mermaid
flowchart TD
    Start([任务启动]) --> Init[初始化后台管理器<br/>background_manager.update_config]
    Init --> CheckTestMode{测试模式?}
    
    CheckTestMode -->|是| SkipWait[跳过场景检测]
    CheckTestMode -->|否| WaitGame[等待进入游戏<br/>_wait_for_game]
    
    WaitGame --> InGame{成功进入?}
    InGame -->|否| Fail([返回失败])
    InGame -->|是| InitCtrl
    
    SkipWait --> InitCtrl[初始化控制器]
    InitCtrl --> InitState[StateDetector<br/>状态检测器]
    InitState --> InitMove[MovementController<br/>移动控制器]
    InitMove --> InitSkill[SkillController<br/>技能控制器]
    InitSkill --> InitDist[DistanceCalculator<br/>距离计算器]
    
    InitDist --> StartDeathMonitor[启动死亡监控线程<br/>start_death_monitor]
    StartDeathMonitor --> LogConfig[输出技能配置信息]
    LogConfig --> MainLoop([进入主循环])
```

## 三、主循环流程

```mermaid
flowchart TD
    Start([主循环开始]) --> CheckExit{检测退出信号?}
    
    CheckExit -->|是| Cleanup[清理资源]
    Cleanup --> StopDeath[停止死亡监控线程]
    StopDeath --> StopMove[停止移动]
    StopMove --> StopSkill[停止技能]
    StopSkill --> End([任务结束])
    
    CheckExit -->|否| CheckBackground[后台模式检查<br/>check_and_auto_pseudo_minimize]
    CheckBackground --> CheckDeath{检测到死亡?<br/>is_death_detected}
    
    CheckDeath -->|是| StopSkill2[停止技能和移动]
    StopSkill2 --> WaitRevive[等待复活 1秒]
    WaitRevive --> Start
    
    CheckDeath -->|否| DetectSelf[自身位置检测<br/>detect_self 15秒超时]
    DetectSelf --> SelfFound{找到自身?}
    
    SelfFound -->|否| LogError[记录帧信息]
    LogError --> RaiseError([抛出异常])
    
    SelfFound -->|是| DetectBattlefield[战场状态检测<br/>get_battlefield_state_detailed]
    DetectBattlefield --> HandleState[处理战场状态]
    HandleState --> Sleep[短暂休息 50ms]
    Sleep --> Start
```

## 四、战场状态处理流程

```mermaid
flowchart TD
    Start([战场状态判断]) --> CheckState{战场状态?}
    
    CheckState -->|NO_UNITS<br/>无单位| NoUnits[情况1: 搜索单位]
    CheckState -->|ALLIES_ONLY<br/>仅有友方| AlliesOnly[情况2: 跟随友方]
    CheckState -->|ENEMIES_ONLY<br/>仅有敌军| EnemiesOnly[情况3: 攻击敌军]
    CheckState -->|MIXED<br/>混合| Mixed[情况4: 优先攻击]
    
    NoUnits --> StopSkill1[停止自动技能]
    StopSkill1 --> MoveLR[左右移动 15秒]
    MoveLR --> FoundUnit1{发现单位?}
    FoundUnit1 -->|是| Start
    FoundUnit1 -->|否| MoveUp[向上移动 10秒]
    MoveUp --> FoundUnit2{发现单位?}
    FoundUnit2 -->|是| Start
    FoundUnit2 -->|否| Timeout([超时终止])
    
    AlliesOnly --> StopSkill2[停止自动技能]
    StopSkill2 --> GetNearestAlly[获取最近友方]
    GetNearestAlly --> MaintainDistAlly[保持距离 100-200px]
    MaintainDistAlly --> Start
    
    EnemiesOnly --> GetNearestEnemy[获取最近敌军]
    GetNearestEnemy --> CheckDist{距离达标?<br/>100-200px}
    CheckDist -->|是| StartSkill[启动自动技能]
    StartSkill --> UpdateSkill[更新技能释放]
    UpdateSkill --> StopMove1[停止移动]
    StopMove1 --> Start
    CheckDist -->|否| StopSkill3[停止技能]
    StopSkill3 --> MaintainDistEnemy[调整距离]
    MaintainDistEnemy --> Start
    
    Mixed --> EnemiesOnly
```

## 五、距离控制逻辑

```mermaid
flowchart TD
    Start([距离判断]) --> CalcDist[计算距离<br/>distance_calc.calculate]
    CalcDist --> CheckRange{距离范围?}
    
    CheckRange -->|距离 < 100px| TooClose[距离过近]
    TooClose --> MoveAway[远离目标<br/>move_away]
    
    CheckRange -->|100px <= 距离 <= 200px| Optimal[最佳范围]
    Optimal --> StopMove[停止移动<br/>movement_ctrl.stop]
    
    CheckRange -->|距离 > 200px| TooFar[距离过远]
    TooFar --> MoveTowards[靠近目标<br/>move_towards]
    
    MoveAway --> End([返回])
    StopMove --> End
    MoveTowards --> End
```

## 六、并行死亡检测机制

```mermaid
flowchart TD
    subgraph 主线程
        A1[主循环] --> A2[快速查询死亡状态<br/>is_death_detected]
        A2 --> A3{检测到死亡?}
        A3 -->|是| A4[停止技能和移动]
        A3 -->|否| A5[继续战斗逻辑]
    end
    
    subgraph 死亡监控线程
        B1[持续循环] --> B2[获取帧]
        B2 --> B3[YOLO检测死亡标签]
        B3 --> B4{检测到死亡?}
        B4 -->|是| B5[设置死亡标志<br/>_death_detected = True]
        B4 -->|否| B6[休眠 50ms]
        B5 --> B6
        B6 --> B1
    end
    
    B5 -.->|共享变量| A2
```

## 七、技能释放逻辑

```mermaid
flowchart TD
    Start([技能更新]) --> CheckEnabled{自动技能启用?}
    
    CheckEnabled -->|否| End([返回])
    CheckEnabled -->|是| CheckAttack{普攻启用?}
    
    CheckAttack -->|是| CheckAttackCD{普攻冷却结束?}
    CheckAttackCD -->|是| DoAttack[释放普攻<br/>按键从热键配置读取]
    DoAttack --> ResetAttackCD[重置普攻计时]
    CheckAttackCD -->|否| CheckSkill1
    ResetAttackCD --> CheckSkill1
    CheckAttack -->|否| CheckSkill1
    
    CheckSkill1{技能1启用?}
    CheckSkill1 -->|是| CheckSkill1CD{技能1冷却结束?}
    CheckSkill1CD -->|是| DoSkill1[释放技能1]
    DoSkill1 --> ResetSkill1CD
    CheckSkill1CD -->|否| CheckSkill2
    ResetSkill1CD --> CheckSkill2
    CheckSkill1 -->|否| CheckSkill2
    
    CheckSkill2{技能2启用?}
    CheckSkill2 -->|是| CheckSkill2CD{技能2冷却结束?}
    CheckSkill2CD -->|是| DoSkill2[释放技能2]
    DoSkill2 --> ResetSkill2CD
    CheckSkill2CD -->|否| CheckUlt
    ResetSkill2CD --> CheckUlt
    CheckSkill2 -->|否| CheckUlt
    
    CheckUlt{大招启用?}
    CheckUlt -->|是| CheckUltCD{大招冷却结束?}
    CheckUltCD -->|是| DoUlt[释放大招]
    DoUlt --> ResetUltCD
    CheckUltCD -->|否| End
    ResetUltCD --> End
    CheckUlt -->|否| End
```

## 八、伪后台模式流程

```mermaid
flowchart TD
    Start([后台模式检查]) --> CheckConfig{后台模式启用?}
    
    CheckConfig -->|否| End([返回])
    CheckConfig -->|是| CheckMinimized{窗口最小化?}
    
    CheckMinimized -->|否| End
    CheckMinimized -->|是| SavePos[保存原始位置]
    SavePos --> RestoreWin[从最小化恢复]
    RestoreWin --> MoveOffScreen[移动到屏幕外<br/>-32000, -32000]
    MoveOffScreen --> SetFlag[设置伪最小化标志]
    SetFlag --> End
```

## 九、异常处理流程

```mermaid
flowchart TD
    Start([异常发生]) --> LogError[记录异常信息]
    LogError --> LogFrame[记录当前帧信息]
    LogFrame --> Cleanup[清理资源]
    Cleanup --> StopMonitor[停止死亡监控线程]
    StopMonitor --> StopMove[停止移动]
    StopMove --> StopSkill[停止技能]
    StopSkill --> Raise[向上抛出异常]
```

## 十一、关键性能指标

| 指标 | 优化前 | 优化后 |
|------|--------|--------|
| 死亡检测频率 | ~10Hz (100ms间隔) | ~20Hz (50ms间隔) |
| 死亡检测方式 | 同步阻塞 | 并行线程 |
| 主循环延迟 | 100ms | 50ms |
| 后台支持 | 需激活窗口 | 支持伪后台 |
