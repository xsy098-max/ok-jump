# ok-jump 项目指南

## 项目简介

漫画群星：大集结自动化工具，基于 ok-script 框架开发。支持自动战斗、自动匹配、自动登录、新手教程、日常任务等功能。

## 版本号规则

每次功能更新或 bug 修复后，自动递增 `config.py` 中的 `version` 字段：
- 小改动（bug 修复、小调整）→ patch +1：`1.4.30` → `1.4.31`
- 大改动（新功能模块、重大重构）→ minor +1：`1.4.30` → `1.5.0`

## 推送打包

用户说"推送打包"时，完整流程：更新版本号 → git commit → git tag → git push，GitHub Actions 自动打包发布。

## 关键文件

| 文件 | 作用 |
|------|------|
| `config.py` | 全局配置（版本号、按键、窗口参数、任务列表） |
| `main.py` | 主入口（正式版） |
| `main_debug.py` | 调试入口（无 GUI） |
| `pyappify.yml` | 打包配置（China + Debug profile） |
| `src/task/` | 各自动化任务 |
| `src/combat/` | 战斗系统（状态检测、技能控制、距离计算） |
| `src/tutorial/` | 新手教程状态机 |
| `src/utils/` | 工具类（截图、后台输入、分辨率适配、繁简转换） |
